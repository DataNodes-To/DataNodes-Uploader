# DataNodes Uploader

Windows desktop app to upload files to DataNodes, replace existing files in
place (the share link stays the same), and organise them into folders.

## Download

Get the latest **DataNodes Uploader.exe** from the
[Releases page](https://github.com/DataNodes-To/DataNodes-Uploader/releases/latest),
or use the always-latest link:

```
https://github.com/DataNodes-To/DataNodes-Uploader/releases/latest/download/DataNodes.Uploader.exe
```

Just double-click the exe and paste your API key. The app updates itself when a
newer build is released.

> This repo hosts the built releases only. The application source is maintained
> privately.
